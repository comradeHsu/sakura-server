日本国立大学大学院 研究生申请专区
日本高水准的教育体制闻名遐迩，在世界各权威大学排行榜单中，日本大学综合实力一直遥遥领先，位居亚洲第一、世界第三。作为日本高等教育最重要组成部分的国立大学，不仅拥有着庞大的优质教育资源，在学术研究水平以及教学环境建设方面均处于顶尖地位。 阅读全部
英文项目申请专区
日本的诸多名校会为海外同学开设全英授课课程。绝大部分全英授课课程可以直接申请到修士。跳过语言学校和研究生阶段。形式上类似申请欧美等其他国家。 G30：G30是2016年之前，由日本政府扶持的13所名校的英文课程的统称。2016年，日本政府取消针对这13所院校的特别扶持，转而将经费更全面的投入到各个学校。因此，虽然G30的名称官方已不再适用，但实质上，日本学校的英文课程是更加丰富和多样了。 阅读全部
语言学校申请专区
日本语言学校是日本为了教育外国籍学生日语而设置的各种语言学校之合称。不同于一般正规教育系统的学校，日本语学校通常不会授予文凭，因此学生就读这些学校，主要的目的都是作为进入日本专门学校、大学或大学院（研究所）的先修教育课程，或其他如求职等用途。在各种统筹管理日本语学校的单位中，日本语教育振兴协会是最重要的一个。 阅读全部
别科申请专区
别科，是指设立在大学内的特别学习科，可以理解为大学内的专门针对外国人的日本语学习科。进入别科的学生和日本学生一起在大学校园内共同学习、生活，可以更深入地了解日本文化。目前，在日本的别科共有48所，全部设立在私立大学之中。 阅读全部
本科申请专区
通常升入日本大学学习本科课程，需要通过日本针对外国人留学生设立的类似高考的日本留学生考试。而日本留学生考试中，日语比重至少一半（总分占一半），且在中国大陆地区并没有设置考场。使得国内绝大部分缺乏日语基础的同学只能选择语言学校赴日后参加考试这一条路。 阅读全部
